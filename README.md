# Viso - 智能交互式视频清理助手 🎥

`Viso` 是一个专为整理庞大视频库设计的高性能命令行工具。它不仅能通过元数据识别垃圾视频，更引入了 **视觉指纹技术**，能够跨分辨率、跨格式揪出重复内容。

## 🌟 核心特性

*   **真正的交互式体验**：支持导览审阅（一键播放预览、并排对比）、批量勾选以及清单编辑模式。
*   **深层视觉比对**：默认 5 点分布采样，支持跨分辨率、跨格式重复检测，支持截取片段识别。
*   **极致安全**：镜像回收站结构，下载中文件保护（5分钟活跃保护），支持跨盘秒级移动。

## 🧹 清理规则详解

`Viso` 内置了一套多维度的逻辑引擎，扫描时会自动应用以下规则：

### 1. 重复检测 (`duplicate`)
*   **物理指纹**：对文件头、中、尾采样计算 MD5。文件大小和采样哈希一致则视为完全重复。
*   **视觉指纹**：均匀提取 N 个时间点的 16x16 灰度缩略图作为指纹。
    *   **异质重复**：解决“同一视频、不同分辨率/格式”的情况。只要视觉指纹高度相似且时长一致（误差 < 3s），即视为重复。
    *   **截取识别**：解决“B 是 A 的一段”的情况。只要视觉指纹序列匹配且 B 显著短于 A，即视为截取片段。
*   **原件判定**：发现重复时，按 **分辨率 > 码率 > 修改时间 > 路径长度** 自动选出最优原件保留，仅清理冗余副本。

### 2. 时长过滤 (`duration`)
*   识别总时长短于阈值的视频（默认 **5s**，通过 `-d` 自定义）。

### 3. 低分辨率过滤 (`resolution`)
*   识别画面过小的视频（默认 **< 480x320**，通过 `-W`, `-H` 自定义）。

## 🛡️ 安全保护逻辑 (自动生效)

为了防止误删，工具默认开启以下保护：
*   **活跃文件排除**：最后修改时间在 **5 分钟内** 的文件自动跳过（防止正在下载/录制）。
*   **临时文件跳过**：自动屏蔽 `.crdownload`, `.part`, `.tmp`, `.aria2`, `.xltd` 等临时后缀。
*   **回收站隔离**：自动跳过所有 `.viso-trash` 文件夹及其子目录。

## 🚀 快速开始

### 1. 安装依赖与编译
需要安装 **FFmpeg**。
```bash
go build -o viso ./cmd/viso/main.go
```

### 2. 开始扫描
```bash
./viso scan [目录]
```

### 常用参数
*   `-s, --samples`: 采样点数量（默认 5）。增加采样点可提升精度，但会减慢扫描速度。
*   `-d, --duration`: 最小保留时长（默认 `5s`）。
*   `-W, --width` / `-H, --height`: 分辨率阈值（默认 `480x320`）。

## 📂 回收站机制
文件永远不会被直接物理删除。它们被移动到扫描根目录下的 `.viso-trash` 文件夹中，并镜像了原有的目录层级。
**彻底删除**：确认无误后，手动执行 `rm -rf .viso-trash`。
